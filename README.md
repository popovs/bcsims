`{bcsims}` is an R wrapper for the BC [Species Inventory Management System (SIMS)](https://sims.nrs.gov.bc.ca/) API. With `{bcsims}`, you can access your species survey data within SIMS and programatically manipulate it. Or, if you already have biodiversity data, you can use `{bcsims}` to prepare data for import into SIMS, rather than doing the upload manually.

This package is still under active development and considered unstable. 

## Installation

You can install the development version of bcsims from [GitHub](https://github.com/) with:

``` r
# install.packages("pak")
pak::pak("popovs/bcsims")
```
